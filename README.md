# 🛡️ Suraksha Guide - Security Training Crossword Game

An interactive, responsive, web-based crossword puzzle game designed for security guard training, facility management staff, and safety awareness programs. 

This application dynamically loads curated security terminology—covering categories like **Fire Safety, Access Control, CCTV, PSARA, First Aid, and Emergency Protocols**—and generates interactive grid puzzles tailored for learners.

---

## ✨ Features

* **Dynamic Grid Generation:** Plays smoothly on a 13x13 auto-adjusting responsive grid suitable for desktop and mobile screens.
* **100+ Specialized Security Terms:** Rich dictionary covering real-world security operations, safety equipment, and emergency procedures.
* **Multi-Category & Difficulty Filtering:** Filter puzzles by Security Guard basics, Fire Safety, Access Control, First Aid, or CCTV.
* **Hint & Clue System:** Detailed, professional training hints provided for every clue across Across/Down directions.
* **LMS & Website Ready:** Easily embeddable on Google Blogger, WordPress, or custom portals via GitHub Pages iFrame.
